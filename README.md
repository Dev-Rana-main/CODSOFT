# CODSOFT Data Science Internship

This repository contains the projects completed as part of the **CodSoft Data Science Internship**.

## Projects

### Task 1 – Titanic Survival Prediction
Developed a machine learning classification model to predict whether a passenger survived the Titanic disaster using passenger information.

**Tech Stack**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

### Task 2 – Movie Rating Prediction with Python
Built a regression model to predict movie ratings based on movie-related features.

**Tech Stack**
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

### Task 3 – Iris Flower Classification
Developed a classification model to identify Iris flower species using flower measurements.

**Tech Stack**
- Python
- Pandas
- Matplotlib
- Scikit-learn
