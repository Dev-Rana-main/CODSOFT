# Titanic Survival Prediction

## Objective
Predict whether a passenger survived the Titanic disaster using Machine Learning.

## Libraries
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Workflow
1. Load dataset
2. Data exploration
3. Handle missing values
4. Encode categorical variables
5. Train-test split
6. Train Random Forest Classifier
7. Evaluate model
8. Visualize confusion matrix and feature importance

## How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
Open `Titanic_Survival_Prediction.ipynb` and run all cells.
